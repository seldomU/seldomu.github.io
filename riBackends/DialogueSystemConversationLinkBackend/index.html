RelationsInspector Node-graph editor for Unity

Dialogue System conversation link backend

Description

This backend displays nodes for all conversations of the scene’s dialogue database. A connection is created where entries of two different conversations are linked.

Usage

Load the scene you want to analyse. Click the Show scene conversations button in the toolbar to generate the graph. Through a conversation’s node context menu you can open the conversation in the dialogue editor, or open a graph of the conversation’s entries. Through the arrow context menu you can open the link’s orgin and destination conversation entries.

Installation

The backend will ship with RelationsInspector soon.
