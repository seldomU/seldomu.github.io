Core GameKit syncro spawner backend

Description

The Core GameKit syncro spawner backend displays syncro spawners, connected to the wave specs they use. Those wave specs are connected to the prefabs or prefab pools they use.

Usage

Load the scene you want to analyse. Click the Show active scene button in the toolbar to generate the graph. The toolbarâ€™s filter options let you see only certain levels or waves.

Installation

The backend is part of the Core Gamekit installation, along with the RelationsInspector Demo.
